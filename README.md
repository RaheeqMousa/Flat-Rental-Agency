# 3lRaheeq Agency - Flat Rental Management System

A comprehensive web-based platform for managing flat rentals, including apartment listings, rental requests, preview scheduling, and customer-owner communications.

## Project Overview

3lRaheeq Agency is a full-featured real estate rental management system that connects property owners with potential renters. The platform streamlines the process of browsing available flats, scheduling previews, submitting rental requests, and managing confirmations.

## Technologies

- **Backend**: PHP
- **Frontend**: HTML, CSS, JavaScript
- **Database**: MySQL/PDO
- **Web Server**: Apache (WAMP64)
  
## Key Features

### User Roles

#### 1. **Customers**
- Browse available flats with advanced search filters
- View detailed flat information, photos, and amenities
- Request flat previews/viewings
- Submit rental applications
- Track rental requests and confirmations
- Receive messages from owners/agency
- Manage personal profile
- View cart

#### 2. **Property Owners**
- Create a Flat
- Receive rental applications
- Accept or reject rental from customers
- Accept or reject flat appointment from customers

#### 3. **Manager**
- Moderate listings and content
- Process confirmations
- List flats in the system

### Core Functionality

1. **Flat Management**
2. **Preview/Appointment**
3. **Rental Process**
4. **Messaging System**
5. **User Management**
6. **Search & Discovery**

## User Flow

### Customer Journey
1. Register/Login → 2. Browse Flats → 3. View Details → 4. Book Preview → 5. Submit Rental Request → 6. Receive Confirmation → 7. View Rented Flats

### Owner Journey
1. Register/Login → 2. Create Flat Listing → 3. Set Preview Times → 4. Receive Applications → 5. Accept/Reject → 6. Confirm Rental → 7. Manage Rentals
